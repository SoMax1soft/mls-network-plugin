## Ключевые особенности

| Функция | Описание |
| :--- | :--- |
| **Удаленная обработка** | Все вычисления выполняются на внешнем сервере анализа. |
| **Модели FAST и PRO** | Два независимых алгоритма детекции. FAST отвечает за мгновенные срабатывания, PRO — за поведенческий анализ. |
| **Межсерверный анализ** | Возможность синхронизации данных о нарушениях между подключенными серверами (если используется общий backend). |
| **Веб-интерфейс** | Панель управления для конфигурации, просмотра логов и статистики. |
| **Уведомления** | Интеграция с Discord и Telegram через вебхуки. |
| **Гибкая кастомизация** | Конфигурация адаптируется под разные режимы игры. |

---

## Системные требования

* **Платформы:** Spigot, Paper и форки (Purpur, Pufferfish и др.)
* **Среда выполнения:** Java 17 или новее

---

## Установка и запуск

1. Загрузите последнюю версию плагина из **GitHub Releases**.
2. Поместите `.jar` в `plugins/` на сервере.
3. Запустите сервер для генерации конфигурации.
4. Настройте конфигурацию и подключение к backend-системе.
